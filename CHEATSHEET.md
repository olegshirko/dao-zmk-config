# Шпаргалка по раскладке Dao42 (macOS)

## Базовый принцип

Каждая клавиша может иметь **два действия**:

- **короткое нажатие** — буква/символ/команда;
- **удержание** — модификатор или слой.

## Домашний ряд (home-row mods)

| Клавиша | Короткое | Удержание |
|---|---|---|
| A | `a` | `Ctrl` |
| S | `s` | `Option` |
| F | `f` | `Shift` |
| J | `j` | `Shift` |
| L | `l` | `Option` |
| ; | `;` | `Ctrl` |

`D` и `K` — обычные буквы, без модификаторов.

### Когда срабатывает модификатор

Поведение `lmt`/`rmt` настроено так, чтобы модификатор не ловился при быстрой печати:

- **Модификатор сработает**, если следующая клавиша нажата **другой рукой**.
- **Модификатор сработает**, если удерживать клавишу дольше `tapping-term-ms` (200 мс).
- **Модификатор НЕ сработает**, если до него была нажата другая клавиша менее 125 мс назад (`require-prior-idle-ms`). Это защита от случайных Ctrl/Option при быстром наборе.

### Практически

- Чтобы набрать `A` и `S` подряд одной рукой — просто печатай быстро, получишь `as`.
- Чтобы сделать `Shift + P` (большая буква правой руки) — зажми `F` (левый Shift) и нажми `P`.
- Чтобы сделать `Ctrl + O` в GoLand — зажми `A` (левый Ctrl) и нажми `O` правой рукой.
- `Ctrl + C` в терминале — зажми `A`, немного подержи (>200 мс), нажми `C` (одна рука). Или используй слой `MAC` (`LWR + O` = Копировать).

## Command на краях

| Клавиша | Короткое | Удержание |
|---|---|---|
| левая верхняя крайняя (`]`) | `]` | `Command` |
| правая верхняя крайняя (`[`) | `[` | `Command` |

- `Cmd + C` — зажми левую крайнюю и нажми `C`.
- `Cmd + V` — зажми левую крайнюю и нажми `V`.
- `Cmd + T`/`Cmd + W` — тоже через левую крайнюю.
- `Cmd + O` — удобнее через правую крайнюю + `O`.

## Слои

| Слой | Как открыть | Что внутри |
|---|---|---|
| `MAIN` | по умолчанию | QWERTY, модификаторы, Command на краях |
| `MAC` | зажать `LWR` (внутренние большие пальцы: `Ret` слева, `Esc` справа) | символы, цифры, macOS-шорткаты (копировать/вставить, Spotlight, Back/Forward) |
| `DEV` | зажать `RSE` (наружные большие пальцы: `Tab` слева, `Del` справа) | терминал, GoLand, стрелки, медиа |
| `ADJ` | из `MAC`/`DEV` через `Tab`/`Del`/`Ret`/`Esc` | Bluetooth, bootloader, reset |

### Важно про слои

- `LWR`/`RSE` — это **удержание**, а не переключение. Отпустил большой палец — слой закрылся.
- Если коротко тапнуть `LWR`/`RSE`, сработает обычная клавиша:
  - левый наружный палец: `Tab` (при удержании — `DEV`);
  - левый внутренний палец: `Ret` (при удержании — `MAC`);
  - правый внутренний палец: `Esc` (при удержании — `MAC`);
  - правый наружный палец: `Del` (при удержании — `DEV`).

## Копировать / вставить

Удобнее всего одной рукой через слой `MAC`:

| Действие | Комбинация |
|---|---|
| Вырезать | `LWR` + `O` |
| Копировать | `LWR` + `I` |
| Вставить | `LWR` + `U` |
| Отменить | `LWR` + `Y` |
| Вернуть | `LWR` + `;` |
| Выделить всё | `LWR` + `[` |

## Терминал

- `Ctrl + C` — зажать `A`, подержать, нажать `C`.
- `Ctrl + D` — зажать `A`, нажать `D`.
- `Ctrl + L` — зажать `A`, нажать `L`.
- `Ctrl + R` — зажать `A`, нажать `R`.
- `Ctrl + Z` — зажать `A`, нажать `Z`.
- `Option + ...` — зажать `S`, нажать нужную клавишу другой рукой.
- Новая вкладка / закрыть вкладку / закрыть окно — слой `DEV`: `Gui+T`, `Gui+W`, `Gui+Shift+W`.

## GoLand

Слой `DEV`:

- `Gui+O` / `Gui+Shift+O` — Go to Class / File.
- `Gui+U` / `Gui+Alt+B` / `Gui+Alt+O` — Super method / Implementations / Go to Symbol.
- `Gui+F` / `Gui+Shift+F` — Find / Find in Path.
- `Gui+Shift+R` — Replace in Path.
- `Gui+Alt+[` / `Gui+Alt+]` — Move to code block start / end.
- `Gui+Alt+L` — Reformat code.
- `Gui+Shift+A` — Find Action.
- Стрелки, Home/End, PgUp/PgDn — правая сторона `DEV`.

## Если что-то не так

- **Вылезает случайный модификатор при печати** — проверь, нет ли паузы >125 мс перед домашним рядом. Чем быстрее печатаешь, тем реже это случается.
- **Не срабатывает `Ctrl + A` одной рукой** — это нормально: `Ctrl` на `A`, а `A` той же рукой. Используй `LWR + [` (Выделить всё) или задерживай `A` дольше 200 мс.
- **Command не срабатывает** — убедись, что удерживаешь крайнюю клавишу, а не просто тапаешь (иначе получится `]` или `[`).
- **Слой не открывается** — `LWR`/`RSE` надо держать, а не тапать.

## Памятка по цветам на картинке

- Розовая подсветка — клавиша, которая **удерживается** для активации слоя (`held`).
- `▽` — прозрачная клавиша (`trans`), наследует поведение из `MAIN`.
