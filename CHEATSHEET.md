# Шпаргалка по раскладке Dao42 (macOS)

## Базовый принцип

Каждая клавиша может иметь **два действия**:

- **короткое нажатие** — буква/символ/команда;
- **удержание** — модификатор или слой.

## Command

`Command` находится на **правом внутреннем большом пальце** (клавиша `Esc`): удерживай её, и она превращается в `Command`. Короткое нажатие — обычный `Esc`.

- `Cmd + C` — зажать правый внутренний палец и нажать `C`.
- `Cmd + V` — зажать правый внутренний палец и нажать `V`.
- `Cmd + T`/`Cmd + W` — зажать правый внутренний палец и нажать `T`/`W`.
- `Cmd + Tab` (переключение приложений) — зажать правый внутренний палец и тапать `Tab` левой рукой.
- `Cmd + Space` (Spotlight) — зажать правый внутренний палец и нажать `Space` левым большим пальцем.

Чтобы не было случайного `Command` при быстром нажатии `Esc`, стоит защита:
- `require-prior-idle-ms = <125>` — если перед `Esc` шла быстрая печать, он остаётся просто `Esc`;
- `tapping-term-ms = <220>` — `Command` сработает только если удерживать клавишу чуть дольше обычного.

## Домашний ряд (home-row mods)

| Клавиша | Короткое | Удержание |
|---|---|---|
| A | `a` | `Ctrl` |
| S | `s` | `Option` |
| F | `f` | `Shift` |
| J | `j` | `Shift` |
| L | `l` | `Option` |
| ; | `;` | `Ctrl` |

`D` и `K` — обычные буквы, без модификаторов.

### Когда срабатывает модификатор

Поведение `lmt`/`rmt` настроено так, чтобы модификатор не ловился при быстрой печати:

- **Модификатор сработает**, если следующая клавиша нажата **другой рукой**.
- **Модификатор сработает**, если удерживать клавишу дольше `tapping-term-ms` (200 мс).
- **Модификатор НЕ сработает**, если до него была нажата другая клавиша менее 125 мс назад (`require-prior-idle-ms`). Это защита от случайных Ctrl/Option при быстром наборе.

### Практически

- Чтобы набрать `A` и `S` подряд одной рукой — просто печатай быстро, получишь `as`.
- Чтобы сделать `Shift + P` (большая буква правой руки) — зажми `F` (левый Shift) и нажми `P`.
- Чтобы сделать `Ctrl + O` в GoLand — зажми `A` (левый Ctrl) и нажми `O` правой рукой.
- `Ctrl + C` в терминале — зажми `A`, немного подержи (>200 мс), нажми `C` (одна рука). Или используй слой `MAC` (`LWR + O` = Копировать).

## Переключение языка

Нажать одновременно `,` и `.` (правая рука, указательный и средний пальцы) — отправляет `Ctrl + Space`, стандартное сочетание macOS для смены источника ввода.

Если у тебя язык переключается через `Cmd + Space`, поменяй в `config/dao.keymap` в блоке `combos` `LC(SPACE)` на `LG(SPACE)`.

## Слои

| Слой | Как открыть | Что внутри |
|---|---|---|
| `MAIN` | по умолчанию | QWERTY, модификаторы, Command на `Space` |
| `MAC` | зажать `LWR` (внутренние большие пальцы: `Ret` слева, `Esc` справа) | символы, цифры, macOS-шорткаты (копировать/вставить, Spotlight, Back/Forward) |
| `DEV` | зажать `RSE` (наружные большие пальцы: `Tab` слева, `Del` справа) | терминал, GoLand, стрелки, медиа |
| `ADJ` | из `MAC`/`DEV` через `Tab`/`Del`/`Ret`/`Esc` | слева — Bluetooth, bootloader, reset; справа — нумпад (цифры и операторы) |

### Важно про слои

- `LWR`/`RSE` — это **удержание**, а не переключение. Отпустил большой палец — слой закрылся.
- Если коротко тапнуть `LWR`/`RSE`, сработает обычная клавиша:
  - левый наружный палец: `Tab` (при удержании — `DEV`);
  - левый внутренний палец: `Ret` (при удержании — `MAC`);
  - правый внутренний палец: `Esc` (при удержании — `MAC`);
  - правый наружный палец: `Del` (при удержании — `DEV`).

## Копировать / вставить

Удобнее всего одной рукой через слой `MAC`:

| Действие | Комбинация |
|---|---|
| Вырезать | `LWR` + `O` |
| Копировать | `LWR` + `I` |
| Вставить | `LWR` + `U` |
| Отменить | `LWR` + `Y` |
| Вернуть | `LWR` + `;` |
| Выделить всё | `LWR` + `[` |

## Терминал

- `Ctrl + C` — зажать `A`, подержать, нажать `C`.
- `Ctrl + D` — зажать `A`, нажать `D`.
- `Ctrl + L` — зажать `A`, нажать `L`.
- `Ctrl + R` — зажать `A`, нажать `R`.
- `Ctrl + Z` — зажать `A`, нажать `Z`.
- `Option + ...` — зажать `S`, нажать нужную клавишу другой рукой.
- Новая вкладка / закрыть вкладку / закрыть окно — слой `DEV`: `Gui+T`, `Gui+W`, `Gui+Shift+W`.

## GoLand

Слой `DEV`:

- `Gui+O` / `Gui+Shift+O` — Go to Class / File.
- `Gui+U` / `Gui+Alt+B` / `Gui+Alt+O` — Super method / Implementations / Go to Symbol.
- `Gui+F` / `Gui+Shift+F` — Find / Find in Path.
- `Gui+Shift+R` — Replace in Path.
- `Gui+Alt+[` / `Gui+Alt+]` — Move to code block start / end.
- `Gui+Alt+L` — Reformat code.
- `Gui+Shift+A` — Find Action.
- Стрелки, Home/End, PgUp/PgDn — правая сторона `DEV`.

## ADJ-слой — цифры

Правая половина `ADJ` превращается в стандартный цифровой блок (как на нумпаде), центрированный под правой рукой:

```
        7   8   9
        4   5   6
    0   1   2   3
```

- средний ряд под пальцами: `4 5 6`
- сверху: `7 8 9`
- снизу: `1 2 3`
- `0` — под `1`, как на обычном нумпаде

Левая половина `ADJ` осталась системной: Bluetooth, bootloader, reset, studio unlock.

## Bluetooth

Всё управление Bluetooth — на левой половине `ADJ`:

| Клавиша | Действие |
|---|---|
| `E` в `ADJ` | `BT 0` — профиль 0 |
| `D` в `ADJ` | `BT 1` — профиль 1 |
| `C` в `ADJ` | `BT 2` — профиль 2 |
| `R` в `ADJ` | `BT 3` — профиль 3 |
| `F` в `ADJ` | `BT 4` — профиль 4 |
| левая крайняя клавиша среднего ряда (`` ` ``) | `BT CLR` — очистить текущий профиль |

Профили `BT 0`–`BT 4` расположены только под **указательным** (`R`, `F`) и **средним** (`E`, `D`, `C`) пальцами левой руки. Сброс профиля (`BT CLR`) остался на своём месте — под мизинцем, чтобы не нажать случайно.

### Как подключить несколько компьютеров и переключаться

1. **Зайди в `ADJ`**: из `MAC` или `DEV` удерживай клавишу с надписью `ADJ` на большом пальце.
2. **Выбери профиль**: нажми `BT 0` — это будет, например, MacBook.
3. **На компьютере**: System Settings → Bluetooth → найди клавиатуру (`Dao Left` или похожее имя) → Connect.
4. **Для второго компьютера**: снова зайди в `ADJ`, нажми `BT 1`, и на втором компьютере подключись к клавиатуре через Bluetooth.
5. **Переключаться между компами**: просто заходи в `ADJ` и нажимай `BT 0` или `BT 1`.

### А если не выбрать профиль перед подключением?

Клавиатура использует **текущий активный профиль** (обычно `BT 0`, если ты ничего не переключал). Если этот профиль пустой — пара создаётся в нём. Если в нём уже записано другое устройство, новый компьютер может не увидеть клавиатуру или отключить предыдущее. Поэтому перед подключением к новому устройству лучше явно выбрать пустой профиль (`BT 1`, `BT 2` и т.д.).

### Если что-то пошло не так

- **Не подключается** — попробуй нажать `BT CLR` на текущем профиле, затем снова `BT 0` (или другой номер) и переподключиться на компьютере.
- **Хочешь сбросить все Bluetooth-пары** — прошей `settings_reset.uf2` (из артефактов сборки) на ту половину, которая подключается к компьютеру.
- **На macOS клавиатура не находится** — убедись, что половина включена и не подключена к другому устройству.

## ZMK Studio

В `dao.conf` отключено блокирование (`CONFIG_ZMK_STUDIO_LOCKING=n`). Это значит, что после прошивки `_studio` варианта ZMK Studio подключится сразу, без необходимости нажимать `studio_unlock`.

## Если что-то не так

- **Вылезает случайный модификатор при печати** — проверь, нет ли паузы >125 мс перед домашним рядом. Чем быстрее печатаешь, тем реже это случается.
- **Не срабатывает `Ctrl + A` одной рукой** — это нормально: `Ctrl` на `A`, а `A` той же рукой. Используй `LWR + [` (Выделить всё) или задерживай `A` дольше 200 мс.
- **Command не срабатывает** — убедись, что удерживаешь правый внутренний большой палец (`Esc`) чуть дольше обычного (~220 мс).
- **Слой не открывается** — `LWR`/`RSE` надо держать, а не тапать.
- **Переключение языка не срабатывает** — проверь в macOS, какое сочетание назначено на смену источника ввода (System Settings → Keyboard → Input Sources).

## Памятка по цветам на картинке

- Розовая подсветка — клавиша, которая **удерживается** для активации слоя (`held`).
- `▽` — прозрачная клавиша (`trans`), наследует поведение из `MAIN`.
